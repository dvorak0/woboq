81/**
82 * Numerically stable function for comparing if floating point values are equal
83 * within epsilon tolerance.
84 * Used for vector and matrix comparison with C++11 compatible functions.
85 *
86 * If either value is NaN or Inf, we check for both values to be NaN or Inf
87 * respectively for the comparison to be true.
88 * If one is NaN/Inf and the other is not, returns false.
89 *
90 * @param check_relative_also is a flag which toggles additional checking for
91 * relative error. This means that if either the absolute error or the relative
92 * error is within the tolerance, the result will be true.
93 * By default, the flag is true.
94 *
95 * Return true if two numbers are close wrt tol.
96 */
97GTSAM_EXPORT bool fpEqual(double a, double b, double tol,
98 bool check_relative_also = true);

200/** compatibility version for ublas' inner_prod() */

206/**
207 * house(x,j) computes HouseHolder vector v and scaling factor beta
208 * from x, such that the corresponding Householder reflection zeroes out

216/**
217 * Weighted Householder solution vector,
218 * a.k.a., the pseudoinverse of the column
219 * NOTE: if any sigmas are zero (indicating a constraint)
220 * the pseudoinverse will be a selection vector, and the
221 * variance will be zero
222 * @param v is the first column of the matrix to solve
223 * @param weights is a vector of weights/precisions where w=1/(s*s)
224 * @return a pair of the pseudoinverse of v and the associated precision/weight
225 */
226GTSAM_EXPORT std::pair<Vector, double>
227weightedPseudoinverse(const Vector& v, const Vector& weights);
