24/**
25 * "Careful" Cholesky computes the positive square-root of a positive symmetric
26 * semi-definite matrix (i.e. that may be rank-deficient). Unlike standard
27 * Cholesky, the square-root factor may have all-zero rows for free variables.
28 *
29 * Additionally, this function returns the index of the row after the last
30 * non-zero row in the computed factor, so that it may be truncated to an
31 * upper-trapazoidal matrix.
32 *
33 * The second element of the return value is \c true if the matrix was factored
34 * successfully, or \c false if it was non-positive-semidefinite (i.e.
35 * indefinite or negative-(semi-)definite.
36 *
37 * Note that this returned index is the rank of the matrix if and only if all
38 * of the zero-rows of the factor occur after any non-zero rows. This is
39 * (always?) the case during elimination of a fully-constrained least-squares
40 * problem.
41 *
42 * The optional order argument specifies the size of the square upper-left
43 * submatrix to operate on, ignoring the rest of the matrix.
44 *
45 *
46 */
47GTSAM_EXPORT std::pair<size_t,bool> choleskyCareful(Matrix& ATA, int order = -1);
48
49/**
50 * Partial Cholesky computes a factor [R S such that [R' 0 [R S = [A B
51 * 0 L] S' I] 0 L] B' C].
52 * The input to this function is the matrix ABC = [A B], and the parameter
53 * [B' C]
54 * nFrontal determines the split between A, B, and C, with A being of size
55 * nFrontal x nFrontal.
56 *
57 * if non-zero, factorization proceeds in bottom-right corner starting at topleft
58 *
59 * @return \c true if the decomposition is successful, \c false if \c A was
60 * not positive-definite.
61 */
62GTSAM_EXPORT bool choleskyPartial(Matrix& ABC, size_t nFrontal, size_t topleft=0);
