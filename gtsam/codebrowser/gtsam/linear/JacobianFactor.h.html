52 /**
53 * A Gaussian factor in the squared-error form.
54 *
55 * JacobianFactor implements a
56 * Gaussian, which has quadratic negative log-likelihood
57 * \f[ E(x) = \frac{1}{2} (Ax-b)^T \Sigma^{-1} (Ax-b) \f]
58 * where \f$ \Sigma \f$ is a \a diagonal covariance matrix. The
59 * matrix \f$ A \f$, r.h.s. vector \f$ b \f$, and diagonal noise model
60 * \f$ \Sigma \f$ are stored in this class.
61 *
62 * This factor represents the sum-of-squares error of a \a linear
63 * measurement function, and is created upon linearization of a NoiseModelFactor,
64 * which in turn is a sum-of-squares factor with a nonlinear measurement function.
65 *
66 * Here is an example of how this factor represents a sum-of-squares error:
67 *
68 * Letting \f$ h(x) \f$ be a \a linear measurement prediction function, \f$ z \f$ be
69 * the actual observed measurement, the residual is
70 * \f[ f(x) = h(x) - z . \f]
71 * If we expect noise with diagonal covariance matrix \f$ \Sigma \f$ on this
72 * measurement, then the negative log-likelihood of the Gaussian induced by this
73 * measurement model is
74 * \f[ E(x) = \frac{1}{2} (h(x) - z)^T \Sigma^{-1} (h(x) - z) . \f]
75 * Because \f$ h(x) \f$ is linear, we can write it as
76 * \f[ h(x) = Ax + e \f]
77 * and thus we have
78 * \f[ E(x) = \frac{1}{2} (Ax-b)^T \Sigma^{-1} (Ax-b) \f]
79 * where \f$ b = z - e \f$.
80 *
81 * This factor can involve an arbitrary number of variables, and in the
82 * above example \f$ x \f$ would almost always be only be a subset of the variables
83 * in the entire factor graph. There are special constructors for 1-, 2-, and 3-
84 * way JacobianFactors, and additional constructors for creating n-way JacobianFactors.
85 * The Jacobian matrix \f$ A \f$ is passed to these constructors in blocks,
86 * for example, for a 2-way factor, the constructor would accept \f$ A1 \f$ and \f$ A2 \f$,
87 * as well as the variable indices \f$ j1 \f$ and \f$ j2 \f$
88 * and the negative log-likelihood represented by this factor would be
89 * \f[ E(x) = \frac{1}{2} (A_1 x_{j1} + A_2 x_{j2} - b)^T \Sigma^{-1} (A_1 x_{j1} + A_2 x_{j2} - b) . \f]
90 */

467 }
